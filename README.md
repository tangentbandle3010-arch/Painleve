## 🧪 Core Topics

現在、以下の3つのテーマを軸に学習を進めています。

### 1. パンルヴェ方程式とガルニエ系
- フックス型微分方程式のモノドロミー保存変形
- リーマン図式（特異点と特性指数）の設定
- ハミルトン構造と正準変数の導入

### 2. 共形場理論 (CFT) の基礎
- 多様体の基礎及びリーマン多様体の周辺
- ヴィラソロ代数 (Virasoro algebra) とその表現
- 中心電荷と共形ブロックの計算

### 3. 両者の接続（最終目標）
- パンルヴェ方程式の「タウ関数」と、CFTの「共形ブロック」の間に成り立つ対応関係の理解。
- 抽象的な理論だけでなく、C++を用いた具体的な数値計算や数式処理による検証も視野に入れる。
